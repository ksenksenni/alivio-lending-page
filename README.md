## Сборка

#### Зависимости

Для разработки нужно:

- NodeJs v12.0.0
- NPM v6.9.0

#### Скрипты

Установить зависимости:

```
 npm i
```

Собрать проект:

```
 npm run build
```

Для разработки используется вотчер:

```
 npm run watch
```
